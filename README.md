# ERC20-based-token
Smart contracts to create ERC20 based tokens (With React Demo). <br>
You can try this smart contract by using localhost or test net (e.g., Ropsten test network)

## Dependencies
- Solidity(Solc-js) 0.6.2
- Truffle 5.4.22 (core: 5.4.22)
- Node.js 14.18.1
- npm 6.14.15
- Web3.js 1.5.3
